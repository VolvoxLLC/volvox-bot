# Security & Privacy

## Data encryption

- **In transit** — All traffic uses TLS 1.3. No unencrypted connections.
- **At rest** — Database and file storage are encrypted using AES-256.

## Authentication

We use Discord OAuth2 for authentication. We never see or store your Discord password. When you log in, Discord confirms your identity and gives us a temporary token.

## Data retention

| Data type | Retention |
|-----------|-----------|
| Audit logs | 90 days |
| Analytics (raw) | 90 days |
| Analytics (aggregated) | Indefinite |
| Moderation cases | Duration of server membership |
| AI conversations | Until deleted by admin |
| Config | Duration of server membership |

You can request full data deletion at any time by contacting support.

## Infrastructure

- **Uptime target** — 99.99% availability
- **Auto-failover** — Automatic recovery from hardware and software failures
- **Monitoring** — 24/7 infrastructure monitoring with alerting
- **Updates** — Security patches applied within 24 hours of disclosure

## GDPR compliance

Volvox.Bot complies with GDPR requirements:

- **Right to access** — Export your data via the API
- **Right to deletion** — Request removal of all personal data
- **Right to portability** — Data available in standard formats
- **Data processing** — All processing occurs in the EU and US

## Reporting security issues

If you discover a security vulnerability, please report it responsibly to security@volvox.bot. We acknowledge reports within 24 hours and aim to resolve critical issues within 72 hours.
