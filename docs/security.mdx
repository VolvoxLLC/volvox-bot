# Security & Privacy

## Data encryption

- **In transit** — Connections to the dashboard and bot use HTTPS/TLS.
- **At rest** — We store data in a managed database, and the hosting provider encrypts it at rest.

## Authentication

We use Discord OAuth2 for authentication. We never see or store your Discord password. When you log in, Discord confirms your identity and gives us a temporary token.

## Data retention

| Data type | Retention |
|-----------|-----------|
| Moderation cases | Duration of server membership |
| AI conversations | 30 days (auto-deleted via cleanup job) |
| Audit logs | 90 days |
| Config | Duration of server membership |

You can request deletion of your data at any time by contacting support.

## Infrastructure

- **Hosting** — Managed infrastructure with automated deployments
- **Updates** — We apply bot and platform updates automatically
- **Recovery** — Automatic restart on failure with health monitoring

## Privacy

We take your privacy seriously:

- We only collect data necessary for the bot to function
- We share data only with service providers essential to running the platform (e.g., Discord for authentication)
- You can request a full export or deletion of your data at any time

For full details, see our [Privacy Policy](https://volvox.bot/privacy).

## Reporting security issues

If you discover a security vulnerability, please report it responsibly to security@volvox.bot. We acknowledge reports within 24 hours and work to resolve issues as quickly as possible.
