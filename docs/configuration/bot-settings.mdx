# Bot Settings

All bot configuration lives in the dashboard under **Config**. No config files, no env vars, no restarts.

## How it works

Every setting is editable through the web UI. Changes are saved and applied immediately — the bot picks up new config within seconds.

<Callout type="tip">
Use the dashboard search to quickly find a specific setting.
</Callout>

## Configuration sections

| Section | What you can configure |
|---------|----------------------|
| **General** | Bot nickname, status, prefix |
| **Moderation & Safety** | Auto-mod rules, content safety model, sensitivity thresholds, actions, and log channel |
| **AI & Automation** | AI chat, triage, and memory |
| **Tickets** | Categories, auto-close, staff roles |
| **Community** | Leaderboards, XP system, public profiles |
| **Temp Roles** | Max duration, allowed roles |
| **Logging** | What events to log, where to send them |

## Dashboard settings

The **Settings** page covers server-level preferences:

- Server-specific display name
- Dashboard access roles
- Notification preferences
- Data export

All settings are per-server — each Discord server you add Volvox.Bot to has its own independent configuration.
