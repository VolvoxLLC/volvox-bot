# API Reference

The Volvox.Bot REST API provides programmatic access to your server data. Use it to build integrations, export data, or automate workflows.

## Base URL

```
https://volvox.bot/api/v1
```

## Authentication

See the [Authentication](/authentication) page for details on API keys and OAuth2.

<Callout type="info">
For full endpoint documentation with request/response schemas, switch to the **API Reference** tab.
</Callout>

## Rate Limiting

All endpoints enforce rate limits. You'll see these headers in every response:

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Max requests per window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Unix timestamp when the window resets |

If you exceed the limit, the API returns `429 Too Many Requests` with a `Retry-After` header.

## Quick examples

**Health check:**
```bash
curl https://volvox.bot/api/v1/health
```

**Authenticated request:**
```bash
curl -H "Authorization: Bearer YOUR_JWT" \
  https://volvox.bot/api/v1/guilds/{guildId}/analytics
```
