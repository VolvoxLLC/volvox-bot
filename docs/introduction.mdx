## Rate Limiting

All endpoints return rate limit headers:

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Max requests per window |
| `X-RateLimit-Remaining` | Requests remaining |
| `X-RateLimit-Reset` | Unix timestamp when window resets |

Community endpoints (leaderboard, showcases, stats, profiles) are rate limited to **30 requests/minute** per IP and require no authentication.

Authenticated endpoints vary by route â€” see individual endpoint docs for specifics.
