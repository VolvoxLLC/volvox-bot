## Bug reports

When reporting a bug, include:

1. **What happened** — The unexpected behavior
2. **What you expected** — What should have happened
3. **Steps to reproduce** — How to trigger the bug
4. **Server ID** — Your Discord server ID (helps us find logs)
5. **Screenshots** — If applicable

You can report bugs via [GitHub Issues](https://github.com/VolvoxLLC/volvox-bot/issues) or in the `#bug-reports` channel on our Discord.

## Feature requests

We love hearing what you want. Submit feature requests via:

- [GitHub Issues](https://github.com/VolvoxLLC/volvox-bot/issues) with the `feature-request` label
- The `#suggestions` channel on Discord

<Callout type="info">
Check existing issues before submitting — your request might already be tracked.
</Callout>
