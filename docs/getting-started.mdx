# Getting started

<Callout type="info">
No installation required. No hosting. No configuration files. We handle everything.
</Callout>

<Steps>

## Add Volvox.Bot to your server

Visit the invite link and authorize the bot for your Discord server:

[Add Volvox.Bot to Discord](https://discord.com/oauth2/authorize?client_id=1467236664853139477&permissions=1099511704582&scope=bot%20applications.commands)

The bot will request only the permissions it needs. Review them and approve.

<Callout type="tip">
You need **Manage Server** permission in the target Discord server to add bots.
</Callout>

## Open the dashboard

Go to [volvox.bot](https://volvox.bot) and click **Login with Discord**. Authorize the OAuth2 connection.

Once logged in, you'll see your server list in the sidebar. Select the server you just added the bot to.

## Configure your server

Open **Config** in the dashboard sidebar. Here you can:

- Toggle features on and off
- Set moderation rules and thresholds
- Configure AI conversation settings
- Set up ticket categories
- Customize community features

<Callout type="tip">
All changes apply instantly — no restarts needed.
</Callout>

## That's it

Your bot is live. Explore the dashboard to see analytics, manage moderation, set up tickets, and more.

</Steps>
