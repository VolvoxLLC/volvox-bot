# Community

Community features help your server members engage and show off their activity. You can toggle every feature from the dashboard.

## What it does

- **Leaderboards** — XP-based leaderboards visible in Discord and on the web
- **Showcases** — Highlight top members or content
- **Public profiles** — Give your members a web-accessible profile page
- **Community hubs** — Read-only public views of server stats and leaderboards

## How to use

1. Enable community features in **Config → Community**
2. Your members earn XP through activity (messages, events, etc.)
3. Leaderboards update automatically
4. View public profiles at `https://volvox.bot/community/{guildId}`

<Callout type="tip">
Community hubs let non-members see your server's public face — great for recruitment.
</Callout>

## Configuration

| Setting | Description |
|---------|-------------|
| Community enabled | Toggle all community features |
| Leaderboard visibility | Public or members-only |
| XP system | Configure how members earn XP |
| Showcases | Which stats to highlight |
| Public profiles | Allow members to have public pages |
