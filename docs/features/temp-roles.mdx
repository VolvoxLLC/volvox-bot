# Temp roles

Assign Discord roles that automatically expire after a set duration. Perfect for giveaways, event access, trial periods, and more.

## What it does

- **Time-limited roles** — Assign a role that the bot removes after a set duration
- **Flexible durations** — Minutes, hours, days, or custom
- **Automatic cleanup** — The bot removes roles on schedule, with no manual tracking
- **History** — Full log of all temp role assignments

## How to use

1. Use the temp role command in Discord: `/temprole assign @user @role duration`
2. The bot assigns the role immediately and removes it automatically when the duration ends
3. View active and expired temp roles in the dashboard under **Temp Roles**

<Callout type="tip">
Use temp roles for event access — grant a role for the event duration and it cleans itself up.
</Callout>

## Configuration

| Setting | Description |
|---------|-------------|
| Max duration | Longest allowed temp role duration |
| Allowed roles | Which roles you can assign temporarily |
| Log channel | Where the bot posts temp role events |
| Who can assign | Required permission level |
