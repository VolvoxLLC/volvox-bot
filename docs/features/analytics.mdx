# Analytics (Live Insight)

Live Insight gives you a real-time view into your server's health and activity. All metrics are available from the dashboard.

## What it does

- **Message volume** — Track messages over time by channel and member
- **Member activity** — See who's active and engagement trends
- **Bot usage** — How often commands are used, AI conversations started
- **Performance** — Bot response times and latency
- **Server health** — Overall server health score

## How to use

1. Open the dashboard → **Overview** for your server to see a quick snapshot
2. Open **Analytics** for detailed charts and metrics
3. Use the date range picker to focus on specific periods
4. Check **Performance** for technical health metrics

<Callout type="info">
Analytics queries support a range of up to 90 days.
</Callout>

## Available metrics

| Metric | Description |
|--------|-------------|
| Messages per day | Total message count over time |
| Active members | Unique members who sent messages |
| Command usage | Bot command frequency |
| AI conversations | Number of AI chat sessions |
| Response time | Average bot response latency |
| Uptime | Bot and service uptime percentage |
