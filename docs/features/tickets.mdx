# Tickets

A built-in support ticket system for your Discord server. Members can create tickets, and your team can manage them from Discord or the dashboard.

## What it does

- **Ticket creation** — Members create tickets via a panel or command
- **Categories** — Organize tickets by type (support, bug, appeal, etc.)
- **Transcripts** — The bot automatically saves a transcript when a ticket closes

## How to use

1. Set up a ticket panel in **Config → Tickets**
2. Members click the button or use the command to open a ticket
3. The bot creates a private channel for the conversation
4. Staff responds and closes when resolved
5. The bot saves the transcript and you can view it in the dashboard

<Callout type="tip">
Set up multiple categories so the bot routes tickets to the right team automatically.
</Callout>

## Configuration

| Setting | Description |
|---------|-------------|
| Ticket categories | Define ticket types and their channels |
| Auto-close | Automatically close stale tickets after N hours |
| Transcript channel | Where transcripts are posted |
| Staff roles | Which roles can see and manage tickets |
| Welcome message | Auto-sent when a ticket is opened |
